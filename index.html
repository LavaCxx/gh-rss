    经过朋友实测的完美中纽双重国籍方案
    WillemNZ: 免责声明 发这篇主题单纯只是为了分享一个路子，这种行为涉嫌违反中国国籍法，如果你决定效仿，一切后果自行承担。 国籍法 根据中国国籍法的规定，中国公民不允许持有多重国籍，取得外国国籍的中国公民自动丧失中国国籍。这条法规很多人都知道，但是在现实生活中，这条法规基本上很难去执行。因为当一个中国公民取得外国国籍后，只要自己不主动申报，国内相关部门很难查知这个人是否取得外国国籍的。 双重国籍出入境面临的问题 虽然国籍法的规定在现实中很难执行，但持有双重国籍的中国国民在出入境中国的时候一般都会遇到问题。对于大部分取得外国国籍的中国公民，一般在入籍之前都是持居住国家颁发的和中国护照绑定的居民签证或者永久居民签证，比如加拿大枫叶卡，美国绿卡，澳洲新西兰 PR 等。在入籍后，由于本国公民无法持有本国的签证，之前的居民签证和永久居民签证会在入籍后或者中国护照过期后失效。签证失效后，如果用中国护照回国，入境的时候还是畅通无阻，但是出境的时候，如果出示中国护照，由于中国护照上没有有效签证，航空公司不能办理登机牌。如果出示外国护照，相当于自助申报了外国国籍，有被边检要求注销中国护照的风险。所以一般双国籍的中国公民回国都会选择绕行第三国。 新西兰的移民法 在 09 年以前，新西兰的移民法是允许永久居民在入籍后保留外国护照的上的永久居民签证。所以 09 年之前，加入新西兰国籍的华人还是可以拿着带有永久居民签证的中国护照回国，出入境不会有任何阻碍。09 年新西兰国会修改了移民法，新的移民法规定新西兰公民不允许持有本国颁发的签证，09 年以后加入新西兰国籍后，中国的护照上的永久居民签证会作废。居民签证作废后，持中国护照入境没问题，但是出境的时候就麻烦了。 公民签注 由于新西兰是个移民国家，很多新西兰公民都具有双重/多重国籍，新西兰政府也知道很多持有双重/多重国籍的公民经常都有持外国护照旅行的需求，所以在 09 年修改移民法之后，新西兰移民局推出了公民签注(citizenship endorsement)，这个公民签注实际上就是一个贴在新西兰公民持有的外国护照上的签证，在新西兰移民局的电子签证验证系统里能查询到，只是因为新的移民法不允许新西兰公民持有本国签证，所以名称上不叫签证(visa)而叫签注(endorsement)。这个签注和国内以前的本式港澳通行证上的签注类似。由于这个签注还是会贴在护照上，所以如果中国公民拿着贴有新西兰公民签注的中国护照回国，还是有被中国边检发现双重国籍的风险。 电子签证和电子公民签注 新西兰前几年开始推行电子签证，由于公民签注本质上和签证是同一种东西，只是名称上不叫签证，所以在推行电子签证之后，公民签注也顺便被电子化了。本质上，电子签证和电子公民签注都只是新西兰移民局数据库里一条和外国护照绑定的记录。因为这个记录只存在新西兰移民局数据库里，所以具有中纽双重国籍的中国公民拿着带电子公民签注的中国护照回中国并不会被中国边检发现具有双重国籍。出境办理登机牌的时候，航空公司是通过新西兰移民局的 API 来查询中国护照持有人是否有入境资格，由于在移民局数据库里有公民签注的记录，所以持中国护照也可以顺利办理登机牌。出境过边检的时候，由于中国护照上没有和双重国籍相关的任何记录，所以边检也会直接放行。 实测 前面提到的方案都是理论，我朋友和我针对这套双国籍方案讨论了好几年，但是我胆小，不敢去实践，现实中身边的人也没听说过有谁去实践这套方案。还好我朋友豁得出去，去年入籍，然后拿中国护照去申请了新西兰公民签注。圣诞假期朋友拿着中国护照回国，走自助通道丝滑入境。前几天朋友从深圳出境回新西兰，出示办理了新西兰公民签注的中国护照顺利值机。过边检直接走自助通道，同样丝滑出境，就这样完美验证了这套双国籍方案。
